# JavaScript Fundamentals
## Per Scholas Module 308 SBA.

Script which takes in course and learner data and outputs grades for each student. The main functionality of this program is written in 
<span style="color: #0f79c8">
*getLearnerData()*
<span />

The function loops over course data, assignment group data, and data for all learners. It uses a hash set to gather all unique learner ids, gathers individual scores for learner submissions, and calculates both individual assignment grades and the overall weighted average. The return value is a array of this data.
